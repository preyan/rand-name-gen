## Description

Name-Gen is a simple and lightweight npm package for generating random names.

## Installation

Install the package with npm:

```sh
npm install random-name
```

## Usage

```javascript
const nameGen = require("random-name");

console.log(nameGen());
```

OR

```javascript
import nameGen from "random-name";

console.log(nameGen());
```
